# DerEnots

## About

DerEnots is a sandbox for creating energy schemes. Create schemes using energy blocks, buttons, delays, lamps, and splitters, control energy flows, and monitor their operation in real time.

Quick launch is available at the link: [Game](https://neon32eeee.github.io/DerEnots/)

## Blocks

### Energy Block

- Designed to transfer energy and can have 4 directions: south, west, north, east.

### Splitter

- Splits the signal into 3 directions, opposite the direction from which the signal came.

### Button

- Designed to simply generate energy for further use.

### Delay

- When a signal is received, it holds the signal for a number of ticks (from 1 to 255).

### Lamp

- Lights up when a signal is received.
